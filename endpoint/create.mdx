---
title: "Find labels for an address"
api: "GET https://www.walletlabels.xyz/api/query"
description: "This endpoint allows you to retrieve labels for a given address. You can specify an address or label query and a limit on the number of results returned. The endpoint uses a search index to search for matching labels, based on the query string. It is updated periodically, so it may take a few minutes for new labels to be searchable."
---

### Body

<ParamField body="query" type="string">
  The address query or label name.
</ParamField>

<ParamField body="limit" type="integer" optional>
  The maximum number of labels to return (default is 20, maximum is 100).
</ParamField>

### Response

<ResponseField name="data" type="array">
  A list of labels that match the given address query.

  <Expandable title="Label Object">

<ResponseField name="address_name" type="string">
  The name of the address for the label.
</ResponseField>

{" "}

<ResponseField name="address" type="string">
  The address for the label.
</ResponseField>

{" "}

<ResponseField name="label_type" type="string">
  The type of label (e.g. token, exchange, etc.).
</ResponseField>

{" "}

<ResponseField name="label_subtype" type="string">
  The subtype of label (e.g. token_contract_address, exchange_name, etc).
</ResponseField>

{" "}

<ResponseField name="label" type="string">
  The name of the label.
</ResponseField>

  </Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://www.walletlabels.xyz/api/query?query=bitcoin&limit=5' \
--header 'Content-Type: application/json'
```
